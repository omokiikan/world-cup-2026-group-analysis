# ⚽ 2026 FIFA World Cup – Group of Death Analysis
## 📌 Project Overview
The term **"Group of Death"** is one of the most debated topics during every FIFA World Cup. It is commonly used to describe the group perceived to be the most difficult, often based on opinions, media narratives, or fan discussions.

This project takes a **data-driven approach** to identifying the true Group of Death in the 2026 FIFA World Cup. Using official FIFA rankings, tournament match results, and performance metrics, the analysis evaluates each group based on strength, competitiveness, and overall performance.

The project demonstrates the complete analytics workflow—from data collection and preparation to visualization and insight generation—using Microsoft Excel and Power BI.

---

## 🎯 Project Objectives

This project aims to answer the following questions:

- Which group has the strongest collection of teams?
- Which group is the most competitive based on match outcomes?
- Which group scores the most goals?
- Which group is the most balanced in terms of team quality?
- Which group deserves to be called the **Group of Death** based on data rather than opinion?

---
## 📊 Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

- 🏆 Toughest Group (Highest Death Index)
- ⭐ Strongest Group (Lowest Average FIFA Ranking)
- ⚽ Highest Scoring Group
- 📈 Highest Average Goals per team
- 🤝 Most Competitive Group
- ⚖️ Most Balanced Group (Smallest Ranking Spread)
- 🚨 Biggest Tournament Upset

## 🧮 Custom Death Index

To evaluate each group objectively, a custom **Death Index** was developed.

The score combines multiple indicators of group difficulty:

- Average FIFA Ranking
- Ranking Spread
- Average Goal Difference
- Average Goals per Match

Groups are ranked based on the combined score to determine which one is statistically the most difficult.
